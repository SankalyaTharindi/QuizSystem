Problem: "Updates were rejected"
Solution: git pull origin main --rebase
          git push origin main

Problem: "Not a git repository"
Solution: Make sure you're in the QuizSystem folder

Problem: Wrong branch name?
Solution: Use 'master' instead of 'main' in commands
          git push origin master

Problem: Want to exclude .class files?
Solution: echo *.class >> .gitignore
          git add .gitignore
          git commit -m "chore: Update gitignore"
